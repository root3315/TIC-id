## ðŸ› ï¸ Reporting a Vulnerability

If you discover a security issue in ExoVision, please follow these steps:

1. **Do not post the vulnerability publicly.**  
   Share it privately to prevent abuse before a fix is released.

2. **Report via email:**  
   - Email: `sozdayvek@gmail.com`  
   - Subject: `[Security] Vulnerability in ExoVision`

3. **Provide details:**  
   - Version of ExoVision affected  
   - Steps to reproduce the issue  
   - Screenshots, logs, or proof-of-concept code (if possible)

4. **Response time:**  
   - You will receive an acknowledgment within **48 hours**.  
   - Security fixes will be released as soon as possible, and you will be notified once patched.

5. **Resolution:**  
   - If accepted, a fix will be issued, and the reporter may optionally be credited in release notes.  
   - If declined, a clear explanation will be provided.
